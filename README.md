# Setup
```bash
    # Clone repository
    git clone https://github.com/tomas6446/nosql-mongodb

    # Enter the directory
    cd nosql-mongodb

    # Run Postgres container
    sudo docker-compose up -d
    
    # run project on Unix-like systems
    ./gradlew bootRun  
```
